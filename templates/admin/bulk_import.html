Import Guidelines
File Requirements:
  • CSV format only
  • UTF-8 encoding
  • First row must contain headers
  • Maximum file size: 10MB
Data Validation:
  • Required fields must be provided
  • Email addresses must be unique
  • Boolean values: true/false
  • Duplicate entries will be skipped
